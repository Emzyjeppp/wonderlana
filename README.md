# WonderLana

Interactive skeuomorphic "desk workspace" web application honoring Aurhel Alana (Lana JKT48) for her 100th Show Milestone and her 20th Birthday Seitansai. Inspired by the Framer "Desk" template by Cray.

## Features

- **Interactive Desk Surface**: A styled dark grid workspace where elements can be dragged and repositioned.
- **Draggable Elements**:
  - **Lana Polaroid Photo**: Draggable portrait photo of Lana JKT48 (`daidaidai.jpg`).
  - **Vinyl CD Player**: Clickable vinyl widget that spins on toggle.
- **Dual Folder Drawers**:
  - **100th Show Folder**: Opens the bottom drawer presenting the *7 Wonders of the 100th Show* (Kawaii themed).
  - **Seitansai Folder**: Opens the bottom drawer presenting the *7 Wonders of the 20th Birthday* (Mature themed).
- **Zero Dependencies**: Lightweight layout built with raw HTML, CSS, and Vanilla JS in a single unified entry point (`index.html`).

## Installation & Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/<username>/wonderlana.git
   ```
2. Open `index.html` directly in any web browser.
